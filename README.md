# ゲームアルゴリズム 前期末課題試験 in C++ / DXLib / JetBrains CLion

## この作品について

この作品は横浜デジタルアーツ専門学校の授業 **「ゲームアルゴリズム」** の前期末試験の一環として作成された作品です。

## 前提ソフトウェア
- 開発には`JetBrains CLion 2023.2.1`を使用しています。
- DxLibは自動インストールになっています。特別な設定は不要なはずです。
- ビルド設定は手動設定です。なお当環境ではビルドにMinGWではなくVisual Studio 2022を推奨しております。

## この課題の要件

### 以下の項目を満たす作品を制作する。
- アームの移動操作が可能
- ワイヤーの伸縮操作が可能
- アームの操作と連動してふりこが振幅する

※上記の要件を満たせば、ゲームの面白さや演出の良し悪しは問いません。
※ただしエンジン側が提供する物理処理の機能やアセット、有料・無料のプラグインを使用した場合は失格とします。

### 期限
９月８日（金）１５：３０まで

## 操作方法

|  キー   |      操作       |
|:-----:|:-------------:|
|   W   |  マグネットを短くする   |
|   S   |  マグネットを長くする   |
|   ↑   |  アームを上に移動する   |
|   ↓   |  アームを下に移動する   |
|   →   |  アームを右に移動する   |
|   ←   |  アームを左に移動する   |
| Enter | ゲームクリア時に終了できる |

## ソースコードに関して

- ソースコードは`src`フォルダにあります。
- 今回は`main.cpp`と`vector.h`のみで構成されています。
